# VNL 2023 — Sports Performance Analytics (EDA Report)

## Project Summary

A comprehensive exploratory analysis on the Volleyball Nations League 2023 dataset to identify competitive patterns, player efficiencies, and winning indicators. The goal is to derive actionable insights using statistical techniques and visual data storytelling suitable for analytics-driven decision frameworks in sports.

---

## Objectives

* Assess team and player performance consistency
* Analyze scoring distribution and match outcome dependencies
* Examine the correlation between key performance KPIs
* Identify top contributors and strategic performance gaps
* Present insights visually for rapid analysis and reporting

---

## Data Assets

| File Name     | Purpose                                         |
| ------------- | ----------------------------------------------- |
| `VNL2023.csv` | Official VNL 2023 match & performance dataset   |
| `note.ipynb`  | Complete EDA workflow including visual insights |

All records were subjected to data cleaning, standardization, handling missing values, and structural validation.

---

## Key Analytical Findings

* Distinct performance concentration among top-ranked teams
* High positive correlation between attack efficiency and victory rate
* Blocking and serving stats impact outcomes differently across teams
* Variations in player contributions indicate strategic specialization

*Visual analytics and conclusions are included in the notebook.*
